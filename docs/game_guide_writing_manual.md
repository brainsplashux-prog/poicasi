# ゲーム攻略記事 執筆マニュアル v1.0

> **このマニュアルは必ず参照してから記事を書くこと。**
> ターゲット：ゲーム未経験者・ポイ活初心者。「そのゲームを一度もプレイしたことがない人が読む」前提で書く。

---

## 1. 記事の選定基準

| 条件 | 基準値 |
|------|--------|
| 最高報酬 | 5,000円未満 |
| 達成日数 | 7日以内 |
| 時給効率ランク | S（800円/h以上）A（500〜799）B（300〜499）|
| 難易度 | ★☆〜★★★☆☆（初心者が無課金で達成できる） |

---

## 2. 記事の必須構成（この順番で書く）

```
1. game-summary-box（攻略サマリー）
2. この記事でわかること（1〜2行の結論）
3. ⚠️ 警告ボックス（ポイントサイト経由のみ等）
4. ポイントサイト別 報酬比較表
5. ゲーム概要（どんなゲーム？）
6. 画面の基本構成（インストール直後に覚えること）
7. 最初にやること（文明選択・初期設定等ゲーム固有の重要事項）
8. 日別攻略ロードマップ（day-roadmap）
9. よくある質問 Q&A（faq-item × 4〜5問）
10. まとめ（3〜4か条）
11. cta-box（ポイカジCTA）
```

---

## 3. 絶対に守る文章ルール

### ❶ 操作は「パス形式」で書く
**NG例：** 「農場を建設してください」  
**OK例：** 「画面下メニューの『建設』→『農業』→『農場』をタップ → 空き地に配置する」

### ❷ スタミナ・AP・エネルギーは必ず数値で書く
- 最大値・回復速度（例：5分に1回復）・全回復時間を明記
- 1日何回プレイすれば効率的かを具体的に書く
- 例：「スタミナは最大140。5分に1回復なので全回復まで約11時間40分。朝・昼・夜の3回プレイで使い切るのがベスト」

### ❸ 「なぜするのか」を必ずセットで書く
**NG例：** 「同盟に入りましょう」  
**OK例：** 「同盟に入りましょう。理由：メンバーから建設時間を短縮する『ヘルプ』を受けられるため、最大50%の時間短縮ができます」

### ❹ 専門用語には必ず補足を入れる
初出の専門用語には括弧で説明を加える。  
例：「政庁（城の中心となる建物）」「スタミナ（バトルに消費するエネルギー）」

### ❺ Q&Aは「ユーザーが実際に詰まる場面」を想定する
- 「〇〇できません」「〇〇が足りません」「課金しないと無理ですか」
- 「ポイントがいつ付きますか」は必須で入れる

## 4. 日別ロードマップの書き方

```html
<div class="day-roadmap">
<div class="day-roadmap__title">〇〇日間の完全攻略ステップ</div>

<div class="day-step">
  <div class="day-step__num">1</div>
  <div class="day-step__content">
    <h4>Day 1（プレイ時間：約〇時間）：〇〇〜〇〇</h4>
    <!-- 以下のセクションを必ず含める -->
    <!-- ◆ 具体的な操作手順（パス形式） -->
    <!-- ◆ その日の終わりにやること（翌日の準備） -->
  </div>
</div>
```

- Day ごとに「プレイ時間の目安」を記載する
- 「寝る前にやること」「朝起きたらやること」を必ず書く
- 最終Dayは「達成確認の方法」で締める

## 5. スタミナ管理テンプレート

```
【〇〇のスタミナ（AP）仕様】
- 最大値：〇〇
- 回復速度：〇分に1回復
- 全回復時間：約〇時間
- 推奨プレイサイクル：朝 / 昼 / 夜の3回
- 回復アイテムの使いどき：残り2〜3日で足りない場合に使用
```

## 7. 難易度別の注意点

| 難易度 | ★★☆☆☆ | ★★★☆☆ | ★★★★☆ |
|--------|--------|--------|--------|
| プレイ時間の目安 | 30分/日 | 1〜1.5時間/日 | 2時間以上/日 |
| スタミナ管理 | シンプル | 朝昼夜3回 | 複数スタミナ管理 |
| 追加の注意点 | なし | デイリーミッション必須 | 同盟・ギルド必須 |

## 8. 必ず入れる固定Q&A（全記事共通）

1. **課金しないと無理ですか？** → 無課金で達成可能な旨を明記
2. **ポイントはいつ付きますか？** → 1〜3週間・管理画面で「未確定」表示になる旨
3. **スタミナの回復時間は？** → 数値で明記
4. （ゲーム固有の詰まりポイントを2〜3問追加）
